Tests retrieving nested text using textContent.

For this test to pass the text "Not changed yet!" in red must be replaced by the text "Rect Text and Rect Animate Text!" in green.

textContent is used in the following ways:

  • to retrieve the text nested within the rect element "sourceTextRect"
  • to set the text in the "originalText" text element to ""
  • to set the text in the "destinationText" text element to the text retrieved from the rect element "sourceTextRect"
